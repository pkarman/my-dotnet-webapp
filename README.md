# Learning .NET

Some notes on learning .NET coming from the Ruby On Rails world.

I worked on MacOS with PostgreSQL running in a local Docker container.

| Rails | .NET |
| Ruby | C# |
| ActiveRecord | Entity Framework Core |
| ERB | Razor Pages |
